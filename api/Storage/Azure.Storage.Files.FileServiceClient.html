Class FileServiceClient

The FileServiceClient allows you to manipulate Azure Storage service resources and shares. The storage account provides the top-level namespace for the File service.

SetProperties(FileServiceProperties, CancellationToken)

The SetProperties(FileServiceProperties, CancellationToken) operation sets properties for a storage account’s File service endpoint, including properties for Storage Analytics, CORS (Cross-Origin Resource Sharing) rules and soft delete settings. You can also use this operation to set the default request version for all incoming requests to the File service that do not have a version specified.

For more information, see https://docs.microsoft.com/en-us/rest/api/storageservices/set-file-service-properties.
