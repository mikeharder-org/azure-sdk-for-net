Class EventHubRetryPolicy

An abstract representation of a policy to govern retrying of messaging operations.

Inheritance
System.Object
EventHubRetryPolicy
Namespace: Azure.Messaging.EventHubs
Assembly: Azure.Messaging.EventHubs.dll

Remarks

It is recommended that consumers not implement custom retry policies but instead configure the default policy by specifying the desired set of retry options when creating an EventHubClient.
